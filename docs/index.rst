
Conocimientos previos
----------------------
El sistema está completamente desarrollado en `Python
<https://www.python.org/>`_ con el framework `Django
<https://www.djangoproject.com/>`_. Algunos de los conceptos
utilizados incluyen el ORM, `Señales
<https://docs.djangoproject.com/en/5.0/topics/signals/>`_, `Vistas
basadas en clases
<https://docs.djangoproject.com/en/5.0/topics/class-based-views/>`_,
`Mixins para vistas
<https://docs.djangoproject.com/en/5.0/topics/class-based-views/mixins/>`_,
y `Grupos y permisos
<https://docs.djangoproject.com/en/5.0/topics/auth/default/#permissions-and-authorization>`_. Para
el frontend, se utiliza únicamente HTML y `Bootstrap 5 completo
<https://getbootstrap.com/docs/5.0/getting-started/introduction/>`_
(javascript).

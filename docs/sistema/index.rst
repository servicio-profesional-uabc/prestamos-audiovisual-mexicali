Sistema PEMA
============
**PEMA** es un software diseñado para gestionar eficientemente el proceso
de préstamos de material de grabación audiovisual en la Facultad de Artes,
campus Mexicali. El software tiene como objetivo proporcionar una
plataforma que permita a alumnos, profesores y personal administrativo
realizar solicitudes, autorizaciones y seguimiento de préstamos de manera
efectiva.
